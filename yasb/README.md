# Install YASB

- Github repo -> [YASB](https://github.com/amnweb/yasb)

- Using Winget install
```shell
> winget install --id AmN.yasb
```

- copy [config.yaml](./config.yaml) and [styles.css](./styles.css) to `C:\Users\[USER]\.config\yasb`

- Wallpapers
Your wallpapers should be stored in: `"%USERPROFILE%\\.config\\Wallpapers"`, you can put any images you like in this folder.
